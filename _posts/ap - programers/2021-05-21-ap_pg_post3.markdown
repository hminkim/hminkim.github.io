### 입출력 예시  

|n|return|
|---|---|---|
|12|28|
|5|6|  
  
  
### 내가 적은 코드

```python
def solution(n):
    sum = 0
    for i in range(1,n+1):
        if n%i == 0:
            sum += i
    return sum
```

### 풀이 과정  

반복문을 통해서 1부터 n까지의 수 중 `n`을 나누었을 때 나머지가 0이 되는 수들을 `sum`에 더하여 `sum`을 출력한다.  
  
### 베스트 코드

```python
def sumDivisor(num):
    return sum([i for i in range(1,num+1) if num%i==0])
```
