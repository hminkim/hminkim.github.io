### 입출력 예시  

|nums|return|
|---|---|
|3|"Odd"|
|2|"Even"|  

  
### 내가 적은 코드

```python
def solution(num):
    if num%2 == 1:
        answer = "Odd"
    else:
        answer = "Even"
    return answer
```

### 풀이 과정  

`num`을 2로 나누어 나머지가 1이면 홀수인 `Odd`출력, 이외는 짝수인 `Even`을 출력했다.
  
### 베스트 코드

```python
def evenOrOdd(num):
    return "Even" if num%2 == 0 else "Odd"
```

### 반성

- 난이도 적으로는 제일 쉬운 문제였지만 과연 고민을 덜 하고 풀이 시간을 줄이는 것이 먼저인지, 고민을 좀 더 하더라도 시간복잡도를 줄이는 것이 먼저인지, 고민을 많이 하게 한 문제 인 것 같다.
